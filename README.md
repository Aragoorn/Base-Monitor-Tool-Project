# Base Network Monitor Tool

A lightweight Python-based utility to track real-time Ethereum market data for Base network participants and developers.

## Features
- **Real-time Price Tracking:** Fetches current ETH/USD rates via CoinGecko API.
- **Developer-Friendly:** Designed to be easily integrated into larger on-chain monitoring systems...

## Tech Stack
- Python 3.x
- Requests Library
